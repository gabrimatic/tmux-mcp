# Contributing

Thank you for helping improve TMUX MCP.

## Development

```bash
npm install
npm run check
```

Please keep changes focused and include tests for behavior changes.

## Pull Requests

- Explain the behavior change and why it matters.
- Include the commands you ran.
- Update README or security docs when the MCP surface or safety model changes.
- Do not include secrets, local paths with sensitive data, or generated provenance metadata.
